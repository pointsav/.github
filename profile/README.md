## 🔒 Security Standards: The Triple-Bind
We enforce a strict separation between development and execution:
1. **The Forge (~/developer):** Identity-bound creation.
2. **The Shop Floor (~/production):** Read-only, signed execution.
3. **The Diode:** Unidirectional control flow via `package-link`.
